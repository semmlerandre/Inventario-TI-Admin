Objetivo do Sistema

Desenvolver um sistema web para gestão de linhas móveis corporativas, permitindo controlar:

Linhas telefônicas

Chips (SIM / eSIM)

Aparelhos celulares

Operadoras

Planos

Colaboradores responsáveis

Solicitações de linhas

Número de chamados

Histórico de movimentações

Custos mensais

O sistema deve permitir rastreabilidade completa de cada linha, incluindo:

quem solicitou

quem está utilizando

histórico de alterações

número do chamado relacionado
